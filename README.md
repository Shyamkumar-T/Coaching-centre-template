# Coaching Center Web Template

A modern frontend web template for coaching centers, built to support digital-first education businesses.

## 🚀 Overview

This project provides a complete UI system for a coaching center platform, including student-facing and admin-facing interfaces.
It is designed with scalability and backend integration in mind.

## ⚠️ Note

This is a **frontend-only project**. Backend integration is to be implemented separately.

## ✨ Features

* Course listing and details UI
* Student dashboard interface
* Admission/enquiry forms
* Responsive design (mobile + desktop)
* Clean and modern UI/UX
